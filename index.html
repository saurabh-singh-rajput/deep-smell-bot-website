About

The Code Smell Detector Bot is a GitHub bot that uses machine learning to detect smells in code every time a pull request is created. The bot provides feedback in the form of comments on the pull request and also creates a commit status check to show the results of the analysis. The bot is designed to help developers identify and fix code smells early in the development process, which can help to improve the overall quality of the codebase and reduce technical debt.

Features

  • Uses machine learning to detect code smells
  • Provides feedback in the form of comments on pull requests
  • Creates a commit status check to show the results of the analysis
  • Helps developers identify and fix code smells early in the development process
  • Improves the overall quality of the codebase and reduces technical debt
  • Customizable parameters to suit your project's needs

Usage

To use the Code Smell Detector Bot, simply create a pull request on GitHub. The bot will automatically analyze the code changes and provide feedback in the form of comments on the pull request. The bot will also create a commit status check to show the results of the analysis. Developers can use the feedback from the bot to identify and fix code smells early in the development process, which can help to improve the overall quality of the codebase and reduce technical debt.

Get Started

To start using our bot, simply install it on your Github repository and it will automatically begin analyzing pull requests. You can customize the bot's parameters to suit your project's needs.
